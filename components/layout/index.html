上下布局
  1. Layout: 布局容器，其下可嵌套Header Aside Body Footer以及Layout，可以放置任何容器中
  2. Header: 顶部导航条，只能放置Layout中
  3. Aside: 侧边栏，只能放置Layout中
  4. Body: 主题部分，只能放置Layout中
  5. Footer: 底部容器，只能放置Layout中

左右布局固定侧边栏和头部

给Header和Aside分别添加fixed属性，即可固定顶部或者侧边栏，组件会自动添加相应样式，无需手动添加样式。另外我们可以通过给Aside添加size属性，来控制侧边栏的宽度，给属性会自动应用到子组件Menu中

属性

Header

属性 说明 类型 默认值
fixed 是否固定头部 Boolean false

Aside

属性 说明 类型 默认值
fixed 是否固定侧边栏 Boolean false
size 侧边栏宽度 "large" | "default" | "small" "default"
collapse 是否收起侧边栏 Boolean false
