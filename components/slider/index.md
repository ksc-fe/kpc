# 属性

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| min | 最小值 | `Number` | 0 |
| max | 最大值 | `Number` | 100 |
| value | 当前值，可用`v-model`双向绑定 | `Number` | 0 |
| unit | 单位，用于刻度展示 | `String` | `""` |
| isShowEnd | 是否展示最小值和最大值刻度 | `Boolean` | `true` |
| isShowInput | 是否展示输入框 | `Boolean` | `true` |
| step | 步长 | `Number` | `1` |
