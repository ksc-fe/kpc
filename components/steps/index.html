属性

Steps

属性 说明 类型 默认值
value 标识当前第几步，从0开始 Number undefined
status 步骤条的状态 "normal" | "error" "normal"
type 步骤条样式 "default" | "line" | "simple" "default"
clickable 是否支持快速切换【已完成】的步骤 Boolean false
